Windows 10
O Windows 10 coloca tudo no rumo certo com uma interface que é ideal para a produtividade. Você também terá várias novas melhorias voltadas para o trabalho, incluindo um novo e ótimo aplicativo de assistente pessoal e funcionalidade de área de trabalho virtual.

Descrição
Desfrute de inicializações mais rápidas, de um menu Iniciar conhecido, porém expandido, e de ótimas formas novas de fazer tudo, mesmo em dispositivos diferentes.

O Windows ainda melhor
O Windows 10 foi projetado para acompanhar você perfeitamente de um dispositivo para outro. Ele é rápido e ágil.
Você ainda obtém chat gratuito ou suporte por telefone com pessoas reais. Com o Windows 10, ficou ainda mais fácil realizar atividades complexas.

Simplesmente perfeito
O Windows 10 combina o Windows que você já conhece e adiciona excelentes aperfeiçoamentos que você vai adorar.
Tecnologias como InstantGo1 permitem a inicialização e o reinício rápidos. Além disso, o Windows 10 conta com mais recursos internos de segurança para auxiliar na proteção contra software mal-intencionado.

Preço: R$ 1.997,00

Visual Studio 2017
O Visual Studio 2017 é a edição mais recente do ambiente de desenvolvimento integrado da Microsoft, capaz de oferecer ao desenvolvedor suporte completo para a criação de software para Windows, Mac, Linux, além de estrutura para desenvolvimento web e de aplicativos para Android e iOS.
Bastante completo e robusto, o Visual Studio 2017 amadurece o bom arsenal de recursos disponíveis na versão 2015 do produto e mantem a oferta das edições Community (gratita) e as edições pagas, Professional e Enterprise.

Descrição
Serviços e ferramentas profissionais para desenvolvedores para a criação de qualquer aplicativo para qualquer plataforma.
Colabore com mais facilidade e seja mais produtivo, não importa se você trabalha por conta própria ou com uma pequena equipe.
Seja mais produtivo
Ferramentas de depuração melhoradas, navegação de código, IntelliSense, refatoração e dicas de código economizam tempo e esforço, não importa qual linguagem ou plataforma você esteja usando.
O Code Lens o ajuda a manter o foco no trabalho, mostrando referências de código, alterações ou quem modificou um método por último, diretamente no editor.
Desenvolvimento móvel cinco estrelas
Crie aplicativos para dispositivos iOS e Android. Desenvolva em C#, compartilhe a lógica e o código do aplicativo em todas as plataformas e obtenha experiências de usuário nativas e totalmente personalizadas com o Xamarin.
Ou então use suas habilidades existentes da Web e compile aplicativos híbridos e baseados em navegador que dão suporte a recursos de dispositivo exclusivos com o Apache Cordova.

Preço: R$ 1.997,00
